## Pattern Recognition: Summary vs. Duplicate

**Summary Files (KEEP):**
- Concise executive overview (< 5 KB)
- Contains unique metrics/tables/recommendations
- References detailed reports
- Different target audience
- Provides navigation aid

**Examples:**
- `PEN300_NETWORK_EVASION_SUMMARY.md` ✓ (159 lines, executive metrics)
- `ANTI_FORENSICS_PLUGIN_SUMMARY.md` ✓ (summary format)
- `C2_ANALYSIS_PLUGIN_SUMMARY.md` ✓ (overview document)

**Duplicate Files (ARCHIVE):**
- Original report superseded by "remine" version
- Same content scope as newer version
- Older/incomplete analysis
- Same target audience (implementers)

**Examples:**
- `PEN300_AMSI_DEFENSES_MINING_REPORT.md` → archived (has REMINE)
- `PEN300_LINUX_POSTEXPLOIT_MINING_REPORT.md` → archived (has REMINE)
- `PEN300_NETWORK_EVASION_MINING_REPORT.md` → archived (has REMINE)
